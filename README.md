# AI CareerKit

An AI-powered resume, portfolio and LinkedIn profile builder built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, **shadcn/ui**, **Prisma**, **PostgreSQL** and the **Anthropic Claude API**.

> 🚧 Under active development — features land in incremental batches. Full documentation arrives with the final batch.

## Quick start

```bash
# 1. Install dependencies
npm install

# 2. Configure environment
cp .env.example .env   # then fill in values

# 3. Push the database schema
npm run db:push

# 4. Run the dev server
npm run dev
```

Open http://localhost:3000.
